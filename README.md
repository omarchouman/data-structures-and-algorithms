## Big-O Notation

Big-O describes the worst-case time complexity of an algorithm.

### Cases (Best to Worst)
| Notation | Name |
|---|---|
| O(1) | Constant |
| O(log n) | Logarithmic |
| O(n) | Linear |
| O(n log n) | Linearithmic |
| O(n²) | Quadratic |
| O(2ⁿ) | Exponential |
| O(n!) | Factorial |

![Big-O Chart](images/Big-O.jpeg)

### Basic Rules
- Arithmetic, comparisons, assignments → O(1)
- Loop over n elements → O(n)
- Memory access → O(1) amortized
